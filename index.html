AI food suggestions

Food swaps and meal estimates use the private Cloudflare Worker when configured. The API key is not stored in the app.

How the private Worker works
  1. The app sends food requests to the Cloudflare Worker.
  2. The Worker reads `ANTHROPIC_API_KEY` from Cloudflare secrets.
  3. Claude Sonnet returns a structured estimate or recipe idea.
  4. If the Worker fails or quota is reached, VialWise falls back automatically.

Static websites cannot truly hide a browser-side API key, so keys belong in the Worker secret only.

AI Worker is optional. Built-in suggestions work without a key.
